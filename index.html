Open-source astrostatistics projects

The following is a selection of Tom Loredo's open-source projects hosted at GitHub:

batse5bp
The batse5bp Python package provides persistant local access to BATSE GRB data hosted at the CGRO SSC. It accesses both the BATSE GRB catalogs, and a significant amount of the detailed GRB data (e.g., time series and spectral data). See recent Sphinx documentation. Note: batse5bp has been subsumed into astrodata; see astrodata.
sdss_legacy
This Python package provides persistant local access to photometric and spectroscopic data (including SEDs) for the SDSS Main Galaxy Sample (MGS), the luminous red galaxy (LRG) sample, and the quasi-stellar object (QSO) sample observed during the SDSS-I/SDSS-II Legacy Survey, as provided in SDSS DR17. This package will likely be subsumed into astrodata/
inference
The inference package is a collection of Python modules implementing a variety of methods targeting the statistical inference problems—and the statistical modeling style—of the physical sciences. Please pardon our dust! Inference is being significantly overhauled as part of updating it from Python 2.7 to Python 3. The GitHub-hosted version includes some work-in-progress; the setup.py script installs only the parts passing consistency and unit tests. Modules implementing C and Fortran extensions (for performance) in particular have not yet been ported; they are being rewritten to use the new NumPy API and build tools. API documentation for the Python-2.7 version is available here; tutorial/appication documentation is forthcoming.
stanfitter
A more "Pythonic" interface for the Stan Bayesian probabilistic modeling language, via PyStan2. Initially developed for the XXVI Canary Islands Winter School of Astrophysics, and subsequently extended for use in Cornell University's STSCI 4780 — Bayesian Data Analysis: Principles and Practice (BDA). Public documentation forthcoming; see demos and useage examples in the LabResources repo for the 2022 edition of the BDA course (Lab11, ff.).
CUDAHM
A C++ framework for GPU-accelerated hierarchical Bayesian inference with two-level multilevel levels, using NVIDIA's CUDA library. The motivating applications are cosmic demographics, such as modeling number-size ("log(N)-log(S)") distributions, luminosity functions, initial mass functions, etc., accounting for measurement error and selection effects. A paper describing the methodology and software is in preparation for Astronomy & Computing.
bb1gamma
A Python implementation of the break-by-one gamma distribution, a proper, tractable generalization of the Schechter function; see the accompanying paper in Research Notes of the AAS.
SkyMapUQ
A Python module implementing methods for quantifying uncertainty in a HEALPix sky map describing an object's or event's sky location uncertainty via a posterior probability density function (PDF) on the sky, stored as a probability mass function (PMF) across HEALPix pixels (that is, the sky map stores values of the PDF integrated over each pixel). New quantifications include the expected search effort, and an information theoretic equivalent area.
CASt Summer School Introduction to Bayesian Data Analysis
Lecture slides and supplementary material for the Introduction to Bayesian Data Analysis lectures presented at the 2024 Penn State Center for Astrostatistics Summer School on Statistics for Astronomers.
